✅ Lista de Tarefas (To-Do List)

Um projeto simples e funcional de lista de tarefas, desenvolvido com HTML, CSS e JavaScript, que permite adicionar, marcar e remover tarefas de forma prática.

🚀 Demonstração

Abra o arquivo index.html no navegador para testar a aplicação.

✨ Funcionalidades
Adicionar novas tarefas
Marcar tarefas como concluídas
Remover tarefas da lista
Interface interativa e responsiva
Atualização dinâmica com JavaScript
🛠️ Tecnologias Utilizadas
HTML5 → Estrutura da aplicação
CSS3 → Estilização e layout responsivo
JavaScript (Vanilla) → Manipulação do DOM e interatividade
📂 Estrutura do Projeto

🎯 Objetivo do Projeto

Este projeto foi desenvolvido para:

Praticar manipulação do DOM com JavaScript
Trabalhar eventos (click, submit, etc.)
Criar uma aplicação interativa simples
Melhorar habilidades em front-end.
